HTML

Hypertext Markup Language (HTML) is the sandard markup language for crating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML consstructs, images and other object, such as interactove forms, may be embedded into the rendered page. It provides a means to create structuren documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
